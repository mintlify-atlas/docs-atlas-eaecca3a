# Welcome to Validador de Acceso

Validador de Acceso is a robust Windows Forms desktop application built in C# that provides secure user authentication with intelligent brute-force protection and automatic account lockout mechanisms.

## Overview

This application demonstrates best practices for implementing security features in desktop applications, including:

- **User credential management** with in-memory storage
- **Password-based authentication** with validation logic
- **Brute-force attack prevention** through attempt tracking
- **Automatic lockout mechanisms** after failed login attempts
- **Asynchronous UI handling** to prevent freezing during security timeouts

## Architecture

The application is built using a clean, modular architecture:

- **Usuario** - Entity class representing user credentials
- **GestorAcceso** - Access manager handling authentication and security logic
- **Form1** - Windows Forms UI providing user interaction layer

## Security Model

Validador de Acceso implements a **three-strike lockout system**:

1. Track failed login attempts per session
2. Lock access after 3 consecutive failures
3. Enforce a 5-second timeout before resetting
4. Resume normal operation after lockout expires

<Note>
The lockout mechanism uses asynchronous delays to prevent UI freezing, ensuring a smooth user experience even during security enforcement.
</Note>
